# Концепт лендинга для компании по строительству современных бань

Портфолио-концепт коммерческого лендинга с проектными кейсами, комплектациями, галереей, FAQ и интерактивным калькулятором стоимости.

## Статус проекта

- Бренд, объекты, отзывы, цены и показатели вымышлены.
- Формы работают в демонстрационном режиме и не сохраняют данные.
- Бэкенд намеренно не подключён.
- Изображения оптимизированы в WebP.

## Локальный запуск

```bash
npm install
npm run dev
```

## Проверка сборки

```bash
npm run build
```
